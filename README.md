Gameplay
The game is played on a 3x3 grid. Players take turns marking an empty cell with their symbol (X or O). The first player to get three of their symbols in a row (horizontally, vertically, or diagonally) wins the game.   

Features
Two-player gameplay   
Clear winner detection
Reset button to start a new game
Technologies Used
HTML: For structuring the content of the web page.
CSS: For styling the game board and elements.
JavaScript: For implementing the game logic and interactivity.
